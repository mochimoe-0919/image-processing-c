# Image Processing (C)

大学の実験で作成した画像処理プログラムです。

## 内容
- 平滑化（フィルタ処理）
- 鮮鋭化
- Sobelオペレータによるエッジ検出
- 2値化（pタイル法）
- 輪郭追跡
- 周囲長・円形度の計算
- 円形物体の抽出

## 使用言語
- C

## 概要
濃淡画像および2値画像に対して画像処理アルゴリズムを実装しました。
円形抽出では誤検出を改善するためにパラメータ調整を行いました。

## 工夫した点
- 円形抽出で三角形が誤って選択される問題に対し、円形度の計算方法や周囲長の算出方法を見直して改善した
- 参考書や資料を参照しながら、アルゴリズムの理解と実装を進めた
